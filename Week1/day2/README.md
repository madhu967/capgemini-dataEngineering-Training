# Day 2 - Advanced SQL Queries

## Overview

Day 2 covers advanced SQL techniques including GROUP BY, JOIN operations, ORDER BY, and nested queries for complex data analysis.

## Topics Covered

### 1. **Group By Queries** (Queries 21-25)

- Grouping data by columns
- Calculating totals per group
- Finding averages per group
- Identifying highest values in groups
- GROUP BY with multiple columns

### 2. **Having Queries** (Queries 26-30)

- Filtering grouped results with HAVING clause
- Finding groups with more than N rows
- Filtering groups by aggregate values
- Comparing aggregates across groups
- HAVING vs WHERE clause usage

### 3. **Order By Queries** (Queries 31-35)

- Sorting results in ascending order
- Sorting results in descending order
- Multi-column sorting
- Combining GROUP BY with ORDER BY
- Ordering aggregated results

### 4. **Join Queries** (Queries 36-45)

- INNER JOIN for matching records
- LEFT JOIN for including unmatched rows
- Joining multiple tables
- Self-joins and correlated joins
- Combining data from different tables

### 5. **Nested and Correlated Queries** (Queries 46-55)

- Subqueries in WHERE clause
- Correlated subqueries
- Finding maximum/minimum values
- Comparing to aggregate values
- EXISTS and IN operators

### 6. **Combined Moderate Difficulty Queries** (Queries 56-65)

- Complex queries combining multiple concepts
- Multiple JOINs with GROUP BY and HAVING
- Subqueries with aggregates
- Date filtering with complex logic
- Department and salary comparisons

## Learning Outcomes

By the end of Day 2, you should understand:

- How to group and aggregate data effectively
- How to use HAVING clause for group filtering
- How to sort and order results
- How to combine tables using different types of JOINs
- How to write nested and correlated subqueries
- How to combine multiple SQL techniques for complex analysis
