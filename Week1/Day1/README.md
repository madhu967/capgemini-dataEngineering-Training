# SQL Topics Learned

## Topics Covered

1. **SELECT** - Retrieve all or specific columns from tables
2. **WHERE** - Filter and search for specific data
3. **JOIN** - Combine data from multiple tables
4. **LIKE & Wildcards** - Search patterns in text (%, \_)
5. **String Functions** - LENGTH() for string operations
6. **Date Functions** - YEAR(), MONTH(), DATE_SUB(), CURDATE()
7. **Aggregate Functions** - SUM(), AVG(), MIN(), MAX(), COUNT()
8. **GROUP BY** - Group data by categories
9. **ORDER BY** - Sort results
10. **LIMIT** - Limit number of results returned

Combining data from multiple tables with JOINs

These fundamentals form the basis for more advanced SQL operations and database analysis.
