# Day 1 - SQL Basics

## Overview

Day 1 covers fundamental SQL query concepts and techniques for retrieving and filtering data from databases.

## Topics Covered

### 1. **Basic Queries** (Queries 1-5)

- Simple SELECT statements to retrieve all columns
- Selecting specific columns
- Filtering rows with WHERE clause
- Filtering with JOIN conditions
- Basic data retrieval

### 2. **String Matching Queries** (Queries 6-10)

- Pattern matching using LIKE operator
- Finding names that start with specific characters
- Finding names that end with specific characters
- Searching for names containing specific characters
- Pattern length matching

### 3. **Date Queries** (Queries 11-15)

- Filtering by specific year
- Filtering by month across years
- Date range filtering (before/after dates)
- Recent data retrieval (last 2 years)
- Date-based WHERE conditions

### 4. **Aggregate Queries** (Queries 16-20)

- SUM function for total calculations
- AVG function for average values
- MIN function for minimum values
- COUNT function for counting rows
- Aggregate functions on specific columns

## Files

- `queries.sql` - Contains all SQL queries for Day 1 (Queries 1-24)

## Learning Outcomes

By the end of Day 1, you should understand:

- How to write basic SELECT statements
- How to filter data using WHERE clauses
- How to use pattern matching with LIKE
- How to work with date functions
- How to use aggregate functions (SUM, AVG, MIN, MAX, COUNT)
